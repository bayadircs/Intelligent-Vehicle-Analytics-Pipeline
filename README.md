Intelligent-Vehicle-Analytics-Pipeline
An end-to-end data engineering and machine learning pipeline built in R to analyze vehicle performance metrics, predict operational distances, classify vehicle reliability status, and segment fleet profiles. 
This project transforms raw automotive data into actionable business intelligence for fleet managers and logistics companies

Business Value & ROI
Instead of looking at raw telemetry in isolation, this pipeline provides immediate operational value:
-(Predictive Maintenance: Forecasts vehicle distance capabilities using regression modeling to optimize scheduling.
-(Risk Classification: Automates the identification of vehicle health statuses (Good" vs "Average) to prevent unexpected breakdowns.
-(Fleet Segmentation: Uses unsupervised learning to discover hidden performance clusters, enabling targeted vehicle upgrades and mechanical audits.
